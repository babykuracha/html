Pseudo Element Selector Example

This is an example paragraph demonstrating the use of pseudo elements.

This paragraph uses the ::first-letter pseudo-element to style the first letter.

 The ::before pseudo-element is used to insert content before the content of the selected element. In this case, it adds "Before: " before the paragraph with the class example.  The ::after pseudo-element is used to insert content after the content of the selected element. In this case, it adds " :After" after the paragraph with the class example.  The ::first-letter pseudo-element is used to style the first letter of the selected element. Here, it enlarges and changes the color of the first letter of the paragraph with the class highlight.
